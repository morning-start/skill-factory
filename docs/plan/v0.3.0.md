#### Step 3: 集成到 CI/CD (2d)

添加 GitHub Action workflow：

```yaml
# .github/workflows/metadata-check.yml
name: Metadata Consistency Check
on: [push, pull_request]
jobs:
  check:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Run metadata check
        run: |
          pip install skill-factory-cli
          skill-factory check-metadata --format json > report.json
      - name: Upload report
        if: always()
        uses: actions/upload-artifact@v3
        with:
          name: metadata-report
          path: report.json
```

**验收**: CLI 可执行，CI 中自动运行，输出符合预期格式

### 任务 3: GitHub Actions CI/CD 基础集成 ⭐⭐⭐ P1

**工作量**: 3 天 | **状态**: ⬜ 待开始  

#### Step 1: 创建完整 Workflow (1.5d)

`.github/workflows/ci.yml`:

```yaml
name: Skill Factory CI
on:
  push:
    branches: [main, develop]
  pull_request:
    branches: [main]

jobs:
  lint:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Markdown Lint
        uses: DavidAnson/markdownlint-cli-action@v9
        with:
          config: .markdownlint.json
          
  test:
    needs: lint
    runs-on: ubuntu-latest
    strategy:
      matrix:
        python-version: ["3.9", "3.10", "3.11"]
    steps:
      - uses: actions/checkout@v3
      - name: Set up Python ${{ matrix.python-version }}
        uses: actions/setup-python@v4
        with:
          python-version: ${{ matrix.python-version }}
      - name: Install dependencies
        run: |
          pip install -r requirements.txt
          pip install -r requirements-test.txt
      - name: Run tests
        run: pytest tests/ -v --cov --cov-report=xml
      - name: Upload coverage
        uses: codecov/codecov-action@v3
        
  metadata-check:
    needs: test
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Check metadata consistency
        run: skill-factory check-metadata --strict
```
